# 🏪 7Eleven Crawling Project

세븐일레븐 편의점의 **1+1** 및 **2+1** 행사 상품 정보를 자동으로 수집하여 CSV 파일로 저장하는 파이썬 크롤링 도구입니다.

---

### 📋 주요 기능
* **Ajax 직접 요청 방식**: 브라우저에서 '더보기' 버튼을 직접 클릭하는 대신, 서버 API에 직접 데이터를 요청하여 수집 속도와 정확도를 극대화했습니다.
* **전수 조사**: `intPageSize` 파라미터를 조절하고 페이지 루프를 통해 누락 없이 모든 행사 상품을 수집합니다.
* **데이터 정제**: 가격의 쉼표 제거, 이미지 절대 경로 변환 등 즉시 활용 가능한 데이터로 정제합니다.
* pTab 1: 1+1 행사 상품, pTab 2: 2+1 행사 상품
* intPageSize: 한 페이지당 상품 수 (코드에서는 100000 설정)  =>  + MORE 버튼을 누르지 않고 한 번에 모든 상품을 가져오기 위함
